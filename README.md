# Valock ⚡

An ultra-fast, lightweight, and open-source character instalock utility designed for Valorant. **Valock** helps you secure your favorite agent instantly as soon as the agent selection screen appears.

---

## ✨ Features
* **Blazing Fast:** Optimized pixel/color detection or API-based selection (hangisini kullandıysan seç) to lock agents in milliseconds.
* **User-Friendly UI:** Simple and clean interface for quick configuration.
* **Low Resource Usage:** Runs smoothly in the background without affecting in-game FPS.
* **Customizable:** Easily switch between different agents before queueing up.

---

## 🛠️ Tech Stack
* **Language:** Python 
* **Libraries used:** OpenCV, PyAutoGUI, keyboard, pyautogui

---

## 🚀 Getting Started

### Prerequisites
Before running the project, ensure you have the following installed:
* [Python 3.x](https://www.python.org/) (Eğer Python ile yazdıysan)
* Valorant running in **Borderless Windowed** or **Windowed** mode (Eğer piksel tabanlıysa bu uyarı önemli)
